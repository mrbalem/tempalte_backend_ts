# tempalte_backend_ts

## Setup

1. Go to project folder

> cd tempalte_backend_ts

2. Install Dependencies

> npm run setup

3. Create .env with following content

```
PORT=3000
SECRET=AAA

// firebase config proyect

TYPE=
PROJECT_ID=
PRIVATE_KEY_ID=
PRIVATE_KEY=
CLIENT_EMAIL=
CLIENT_ID=
AUTH_URI=
TOKEN_URI=
AUTH_PROVIDER=
CLIENT_CERT=
```

## Run

1. Start Server

> npm start

2. Open in url or using an Endpoint tester (postman)

> http://127.0.0.1:3000/

> http://localhost:3000/
